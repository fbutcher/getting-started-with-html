NOTES ON IMPORTANT CONCEPTS OF HTML

The World Wide Web

The World Wide Web was created in the 1990's. It is host to approximately 30 billion web pages!
The World Wide Web works by users accessing web pages through at browser. The browsers make requests and retrieves page data from web servers via HTTP "Hypertext Transfer Prototocol."
Servers are computers that store the data on the "backend." What the user sees via the browser is called the "frontend." This video explains in detail how the World Wide Web works.

HTML

HTML - hypertext markup language
HTML was created by Tim Berners-Lee in 1989. This is the language used to write most webpages and is used in conjunction with CSS and JavaScript.
HTML is used to display the text of most webpages it is also good at linking images and other webpages. HTML uses tags or elements.

Tags and Elements

HTML Markup uses tags to display hypertext on the web page the following are some examples of tags.

"br"--"/br" Allows the user to create a break to move text to the next line.
"p"--"/p" A new paragraph is started
"em"--"/em" Italics text
"b"--"/b" To bold text

It is important to note that each tag is within angel brackets instead of quotes.

Inline vs. Block

Block elements create an invisble box within the tags. An example of a block element would be the "p" "/p" tags.
Inline elements such as the "br""/br" tag simply wrap text to the next line.

Computers And Programming

Computers are unable to execute any task on their own. They rely on the user to input instructions before any task can be executed.
For this reason, computers are very unforgiving if the instructions, i.e. the code is not clear and exact. Bad code can result in many hours of problem solving.

Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer writes.
